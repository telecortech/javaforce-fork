jfRecord Camera
===============

Record your camera(and mic) to video files.

Provides an optional preview while recording.
Also includes timelapse and stop motion modes.

CodecPack Required:
To use jfRecord Camera you must install the CodecPack which is available at http://javaforce.sourceforge.net/codecpack.php
You have the option of downloading/installing it the first time you run the program.

Site : http://jfrecordcamera.sourceforge.net

Author : Peter Quiring (pquiring at gmail dot com)

Built with the JavaForce SDK

jfAudio Editor
==============

A multi-track audio file editing suite.  Similar to Audacity.

Audio Effects:
 - Amplify
 - fade in/out

CodecPack Required:
To use jfAudio Editor you must install the CodecPack which is available at http://javaforce.sourceforge.net/codecpack.php
You have the option of downloading/installing it the first time you run the program.

Site : http://jfaudio.sourceforge.net

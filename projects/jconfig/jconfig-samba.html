jConfig : Samba


Configure File Sharing / Active Directory

To join a domain:

  • Enter in your Workgroup/Domain and Realm/FQDN
  • Change security to ADS
  • Click Apply
  • Restart the Samba servers
  • Enter in the PDC name/IP address in the Server field
  • Enter in your admin username/password
  • Click Join

Logon Script:
A logon script can be stored on the server at \\$SERVER\sysvol\logon.sh
Some commands available are:

  • jsmbget smb://$SERVER/SHARE/PATH/FILE - retrieve a file from server
  • jmapshare $SERVER $SHARE [x:] - Map a network share and optionally map to a Wine Drive
  • junmapshare $SERVER $SHARE - Disconnect a network share
Commands will use the logged on users password and domain username.

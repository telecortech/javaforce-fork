STUN Server
===========

STUN/TURN Server:

Only supports a small subset of the STUN/TURN features.
Enough to support jPhoneLite and similar phones.
Supports Binding, Allocate, Refresh and Bind requests.
Binding does not support responses from a different IP (from a different port is supported).

WebSite: http://jfstun.sourceforge.net

Source : http://javaforce.sourceforge.net

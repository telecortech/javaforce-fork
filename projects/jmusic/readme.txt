jfMusic
=======

Tracked music editor.

Supports
  - MIDI recording, and provides a simple MIDI keyboard synth.
  - DLS and WAV files (Library files)

The Library files are loaded from ~/Music/Library and on Windows it loads %WINDIR%\System32\drivers\gm.dls

Visit website for some library files if you don't have any.

WebSite: jfmusic.sourceforge.net
Author : Peter Quiring <pquiring at gmail dot com>

Enjoy!
